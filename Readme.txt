This is a simple Desktop Virtual Assistant using Python that can perform several tasks.
The following packages are to be installed and the directory paths to be changed as per the user needs:

1)Pyttsx3:https://pypi.org/project/pyttsx3/



2)SpeechRecognition:https://pypi.org/project/SpeechRecognition/




3) wikipedia:https://pypi.org/project/wikipedia/



4) webbrowser:https://docs.python.org/2/library/webbrowser.html


5) keyboard:https://pypi.org/project/keyboard/
